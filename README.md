# Twitch autoreg

Auto Registrator twitch accounts

## Installation

1. Download and install Python
2. Install libs
```bash
pip install -r requirements.txt
```

## Run

```bash
py twitch.py
```
- Load proxy via url or chose proxy file, select type
- Select threads and account count
- Click `Start` button
